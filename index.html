Buttons Group

Buttons can be grouped horizontal or vertical.
Default one is "vertical".

Horizontal
<div data-role="controlgroup" data-type="horizontal">
    <a href="#" class="ui-btn">button 1</a>
    <a href="#" class="ui-btn">button 2</a>
    <a href="#" class="ui-btn">button 3</a>
</div>

    <a href="#" class="ui-btn">button 1</a>

</div>

Icon get started

back

There is lots of places where you can use icon.
You can use them in class pattern:

Button as anchor
<a href="#" class="ui-btn ui-icon-name ui-btn-icon-left">button</a>
 example:
<a href="#" class="ui-btn ui-icon-star ui-btn-icon-left">button</a> button
Button as HTML tag
<button class="ui-btn ui-icon-name ui-btn-icon-left">button</button>
 example:
<button class="ui-btn ui-icon-star ui-btn-icon-left">button</button>
Button as INPUT tag
<input type="button" value="button" data-icon="name">
 example:
<input type="button" value="button" data-icon="star">
Collapse
<div data-role="collapsible" data-collapsed-icon="name" data-expanded-icon="name">
    <h4>Collapase</h4>
    <p>Collapsed content</p>
</div>
 example:
<div data-role="collapsible" data-collapsed-icon="arrow-d" data-expanded-icon="arrow-u">
    <h4>Collapase</h4>
    <p>Collapsed content</p>
</div>

Icon positioning

back

There are basically four possible positions of icon:

  • Left
  • Top
  • Right
  • Bottom
Button as anchor
<a href="#" class="ui-btn ui-icon-star ui-btn-icon-position">button</a>
 example:
<a href="#" class="ui-btn ui-icon-star ui-btn-icon-top">button</a> button
Button as HTML tag
<button class="ui-btn ui-icon-star ui-btn-icon-position">button</button>
 example:
<button class="ui-btn ui-icon-star ui-btn-icon-top">button</button>
Button as INPUT tag
<input type="button" value="button" data-icon="star" data-iconpos="position">
 example:
<input type="button" value="button" data-icon="star" data-iconpos="top">
Collapse
<div data-role="collapsible" data-iconpos="position">
 <h4>Collapase</h4>
 <p>Collapsed content</p>
</div>
 example:
<div data-role="collapsible" data-iconpos="top">
 <h4>Collapase</h4>
 <p>Collapsed content</p>
</div>
